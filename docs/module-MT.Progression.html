Progression

MT.Progression(pScaleopt, pTemplateopt)

Generate Chords from a scale and template
The template give to the Progression is an array of integers representing
interval between scale degrees. [1, 4, 5] will build triad chords from the 1st, 4th, and 5th scale degrees.
