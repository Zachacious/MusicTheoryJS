Class Note

A musical note. The primary fields are the semitone and octave. The octave is clamped to the range [0, 9]. Setting the semitone to a value outside of the range [0, 11] will wrap the semitone to the range [0, 11] and change the octave depending on how many times the semitone has been wrapped.
